#### NodeJS

##### 删除文件

```JS
npm install del -D
npm install clean -D
```

##### 复制文件

```JS
npm install --save copy
```

##### 压缩文件

```JS
npm install --save copy
```

##### 反向代理、跨域

```JS
npm i http-proxy-middleware
```

#### Gulp

##### 删除文件

```JS
npm install --save-dev gulp-clean
```

##### 复制文件

```JS
npm i gulp-copy -D
```

##### 压缩文件

```JS
npm i uglify -D
```

##### 合并文件

```JS
npm i gulp-concat -D
```

##### 压缩CSS

```JS
npm install gulp-cssnano --save-dev

npm install --save-dev gulp-cssmin
```

##### 压缩图片

```JS
$ npm install --save-dev gulp-imagemin
```

##### HTML文件压缩

```JS
npm install --save gulp-htmlmin
```

##### SCSS转CSS

.scss文件或者.sass文件不能以下划线为文件名开头，否则会出错

```JS
npm install --save sass gulp-sass
npm install --save  gulp-scss
```

##### 删除文件

```JS
npm install grunt-contrib-clean --save-dev
```

##### 复制文件

```JS
npm i grunt-contrib-copy -D
```

##### 压缩文件

```JS
npm i grunt-contrib-uglify -D
```

##### 合并文件

```JS
npm i grunt-contrib-concat -D
```

##### 压缩CSS

```JS
npm i grunt-contrib-cssmin -D
```

##### 压缩图片

```JS
npm install --save-dev grunt-contrib-imagemin
```

##### HTML文件压缩

```JS
npm install grunt-contrib-htmlmin --save-dev
```
